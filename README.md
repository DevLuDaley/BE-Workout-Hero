# BE-Workout-Hero
BE-Workout-Hero is the backend rails-only api repo supplying data to the Workout Hero App.

## Installation

staart up the rails server and you're ready to go.
```terminal
rails s
```

## Usage
Endpoints:
```
      api/v1/routines
      api/v1/workouts

```
